## 🚀 Featured Projects

### 🤖 AI-Powered Security Tools

**[GPT-Fuzzer](https://github.com/sherdencooper/GPTFuzz)** ⭐ 500+ stars  
Scalable fuzzing framework for assessing LLM safety and jailbreak resistance. Now integrated into **Microsoft Azure's PyRIT security toolkit** for enterprise customers worldwide.

**[PatchAgent](https://github.com/cla7aye15I4nd/PatchAgent)** ⭐ 100+ stars  
An intelligent program repair agent that mimics human debugging workflows. Successfully repaired **10 real-world CVEs** across major open-source projects. *USENIX Security 2025*

**[PortGPT](https://github.com/OS3Lab/patch-backporting)**  
First-of-its-kind LLM-based system for automated patch backporting. Successfully backported 9 Linux kernel patches from mainline to stable versions. *IEEE S&P 2026*

### 🛡️ Low-Level Security & Systems

**[ShadowBound](https://github.com/cla7aye15I4nd/ShadowBound)**  
Novel heap memory protection mechanism using LLVM-based compiler optimizations. Achieves significant performance improvements over existing solutions. *USENIX Security 2024*

**[Qiling Framework](https://github.com/qilingframework/qiling)** ⭐ 5,000+ stars  
Core contributor to this advanced binary emulation framework. Implemented comprehensive MCU emulation supporting STM32, GigaDevice, and SiFive microcontrollers across ARM Cortex-M and RISC-V architectures.

### 🔧 Compiler & Hardware Projects

**[Pymx Compiler](https://github.com/cla7aye15I4nd/Pymx)**  
Complete compiler for Java-like language targeting RISC-V, featuring lexical analysis, parsing, semantic analysis, optimization passes, and code generation.

**[RISC-V CPU Implementation](https://github.com/cla7aye15I4nd/trivial-riscv-cpu)**  
Out-of-order RISC-V processor in Verilog implementing Tomasulo algorithm with instruction cache, load buffer, and branch prediction.
